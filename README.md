## 1. Базовые навыки IO
Делаем одно консольное приложение с разными функциями:

В программе должно быть реализовано меню примерно следующего содержания:
1. Создать файл
2. Список файлов
3. Анализ файла
0. Выход

Третью часть попробуем сделать в виде класса с разными методами анализа:
- сколько символов в файле;
- сколько слов в файле;
- сколько слов без предлогов (в, без, до, из, к, на, по, о, от, перед, при, через, с, у, за, над, об, под, про, для).

Дополнительно можно реализовать работу с аргументами, например, если в качестве аргумента передать имя файла, то сразу переходить к анализу.
